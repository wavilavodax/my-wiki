# Moss 策略设计全指南

## 描述四层（必须包含）
[风险态度]+[方向偏好]+[入场逻辑]+[退出逻辑]+"不做什么"
- 方向: 0-1, 0.5中性, 0.8强多, 0.2强空
- 风险: 杠杆/止损/仓位
- 入场: 趋势确认/波动爆发/均值回归
- 退出: 止损/止盈/持仓多久
- "不做什么": 拒绝死水/不加亏损仓/规避事件窗口/资金费率方向

## 五指标优先级（PnL最不重要）
1. 最大回抽(<25% 可接受，>50%危险)
2. 期望值(胜率×盈亏比，数学必须为正)
3. Sharpe(>1.0稳，<0.5赌博)
4. 摩擦成本
5. PnL

## 健康趋势策略画像
- 胜率: 35-45%
- 盈亏比: 2.5-4:1
- Sharpe > 1.0
- 最大回抽 < 25%
- ATR倍数: 1.5-2.5

## 五维信号权重（自动归一化）
| 参数 | 含义 |
|------|------|
| trend_weight | 趋势跟踪。EMA交叉、Supertrend |
| momentum_weight | 动量。RSI、MACD |
| mean_revert_weight | 均值回归。布林带 |
| volume_weight | 量能。OBV、量价配合 |
| volatility_weight | 波动率。ATR突破/收缩 |

## 核心阈值
| 参数 | 范围 | 含义 |
|------|------|------|
| entry_threshold | 0.05~0.55 | 0.15激进/0.25中性/0.40保守 |
| exit_threshold | 0.03~0.30 | 持仓反向信号超此值平仓 |
| long_bias | 0~1 | 0=只做空, 0.5=双向, 1=只做多 |

## 实盘教训（AMETH v1 亏损归因）
- 移动止盈=利润截断神器（5%激活就把利润吃了）→ 趋势策略必须关
- 40x + 5.5RR + 滚仓复利 = 排行榜冠军配置
- 趋势策略胜率低(35-45%)，靠盈亏比赚钱，止盈不够宽就亏
- composite signal 压制链：ADX<25 × 0.5 + 非强BULL regime × sensitivity → 信号可被削弱至1/4

## 参数调优逛查表
| 用户说 | 调什么 |
|--------|--------|
| 止损太紧/被扫太多 | sl_atr_mult ↑ |
| 止损太松/亏损太大 | sl_atr_mult ↓ |
| 交易太频繁 | entry_threshold ↑ |
| 交易太少 | entry_threshold ↓ |
| 更激进 | base_leverage ↑, risk_per_trade ↑ |
| 更保守 | base_leverage ↓, entry_threshold ↑ |
| 让利润奔跑 | tp_rr_ratio ↑, trailing_enabled=false |
