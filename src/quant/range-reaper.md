# 震荡猎手 (Range Reaper)

均值回归策略，10x杠杆，适应横盘市。2026-05-09 创建。

## 策略定位
- 模式：纯均值回归，布林带触轨+RSI超买超卖识别区间边界
- 杠杆：10x
- 方向：双向 (long_bias=0.5)
- 宗旨：见好就收，不追趋势

## 核心参数
| 参数 | 值 | 含义 |
|------|-----|------|
| trend_weight | 0.05 | 视趋势信号为噪音 |
| mean_revert_weight | 0.40 | 布林带回归 = 核心信号 |
| momentum_weight | 0.20 | RSI超买超卖辅助 |
| volatility_weight | 0.25 | ATR识别区间边界 |
| entry_threshold | 0.18 | 中等阈值 |
| base_leverage | 10 | 留容错空间 |
| tp_rr_ratio | 2.0 | 见好就收 |
| trailing_enabled | false | 不移动止盈 |
| rolling_enabled | false | 不滚仓 |

## 双策略对冲行为矩阵
| 市场状态 | 趋势猎手 | 震荡猎手 |
|---------|---------|---------|
| 强趋势上涨 | 大赚 ✅ | 小亏 |
| 强趋势下跌 | 大赚(做空) ✅ | 小亏 |

核心：无论市场状态，总有一只 Bot 赚钱，另一只最多小亏。
