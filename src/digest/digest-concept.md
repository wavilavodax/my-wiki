# Digest（信息摘要）

> **标签:** #概念 #信息筛选

## 什么是 Digest？

AI 每天帮你筛一遍 newsletter/信息源，生成一篇摘要，让你花最少的时间知道今天有哪些重要内容。

## 为什么需要 Digest？

- 信息源太多，人肉读不过来
- 大部分内容没有价值，但不知道哪些有
- AI 初筛可以过滤噪音，节省时间

## Digest 的两个层次

### 第一层：AI 初筛

让 Agent 读取每天收到的 newsletter 列表，告诉你今天哪篇最值得深读。

### 第二层：Wiki 深度处理

对值得深读的文章，用 LLM Wiki 拆解：写摘要、提取概念、建立双向链接、沉淀进知识图谱。

## LLM Wiki 工作流中的位置

信息源 → Agent 读邮件 → Digest 摘要 → Wiki 深度编译 → 知识沉淀
