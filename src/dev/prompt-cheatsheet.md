# 角色 Prompt 速查

## 最佳资源

**prompts.chat** — 16万 stars 开源 Prompt 库，支持 ChatGPT/Claude/Gemini/Llama
- 🌐 https://prompts.chat/prompts
- 📄 https://github.com/f/prompts.chat (PROMPTS.md)
- 🤗 HuggingFace 数据集：huggingface.co/datasets/fka/prompts.chat

## 何时用

需要特定角色时，直接来这里找：
- 法律：lawyer, legal advisor, attorney
- 财务：accountant, tax advisor, financial analyst
- 写作：novelist, screenwriter, copywriter
- 编程：code reviewer, debugger, architect
- 教育：teacher, tutor, exam prep

## 使用方式

1. 找到合适的 prompt
2. 复制并根据自己场景微调
3. 粘贴到 AI 对话开头

## 注意

这是单一角色调用，适合**一次性任务**。
多智能体团队（Profiles）适合**长期专业化分工**，不要混用。
