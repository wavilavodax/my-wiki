# 双向链接

> **标签:** #概念 #知识管理

## 什么是双向链接？

一个页面引用另一个页面时，被引用的页面也会自动显示"被哪些页面引用"。

## 为什么重要？

传统的网页链接是单向的——你点过去就回不来了。

双向链接让你顺着线索反向追溯，形成知识网络。

## 在图谱视图里的表现

- 点 = 知识页面
- 线 = 双向链接关系

页面越关联，线条越密，知识结构越清晰。

## LLM Wiki 的核心机制

Claude Code 录入文章时，会自动识别并建立双向链接：
- 提到的人物建页 → 自动链回这篇文章
- 提到的工具建页 → 自动链回这篇文章
