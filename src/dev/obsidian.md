# Obsidian

> **标签:** #工具 #知识管理

## 什么是 Obsidian？

本地知识库工具，核心是双向链接和图谱视图。

## 在 LLM Wiki 工作流中的角色

- 作为 Vault（知识库文件夹）的载体
- 提供图谱视图，直观展示概念间的关联
- 支持 Markdown 格式，天然兼容 Wiki 结构

## 核心功能

- 双向链接 页面名 — 概念之间互相引用
- 图谱视图 — 一眼看到知识网络结构
- Local REST API 插件 — 让 AI Agent 能写入文件

## Vault 结构（LLM Wiki 标准）

vault/
├── index.md — 入口索引
├── log.md — 操作日志
├── raw/ — 原始素材
├── wiki/ — AI 整理后的知识页面
└── .obsidian/ — Obsidian 配置
